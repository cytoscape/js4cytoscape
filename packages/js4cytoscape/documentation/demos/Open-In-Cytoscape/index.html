How to add Open in Cytoscape button in web page

Prerequisites

  • An installed and running version of Cytoscape.
  • An internet connection.
    •

    Step 1

    Clone js4cytoscape repository from GitHub, the GitHub source for the js4cytoscape is available at https://github.com/cytoscape/js4cytoscape

    Step 2

    You will find main.js file in the dist folder, and you can copy main.js into your web page folder.

    Step 3

    Copy the following code, and paste it in your html file.



    <type="text/javascript", src="/Users/yxin/git/cytoscape/js4cytoscape/dist/main.js"></script>

    <script type="text/javascript">

    window.onload = checkCytoscape();

    </script>

    <button class="cytobutton", id="cytobutton", title="Cytoscape is connected">Cytoscape is connected</button>

    Step 4

    You will see a button in your web page, if the Cytoscape is installed and opened on your machine, the button will show "Cytoscape is connected".

    If the Cytoscape desktop application is not installed and opened, the button will show "Download Cytoscape".
